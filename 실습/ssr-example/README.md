# SSR-Example

## Introduction

`react-dom`의 `renderToString`과 `renderToNodeStream`을 활용하여 리액트 서버사이드 애플리케이션을 간단하게 만들어보았습니다. 학습용으로 제작되어 많은 부분이 생략되었으며, 절대로 프로덕션 서비스에 사용해서는 안됩니다.

## How to Start

1. `npm install`
2. `npm run build`
3. `npm run start`
4. 다음 페이지에서 서버사이드 렌더링을 확인해보세요.
   - http://localhost:3000/
   - http://localhost:3000/stream
